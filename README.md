Clone the repository

git clone https://github.com/haripriya678/tableau_dashboard.git


Open the Tableau dashboard

Install Tableau Desktop or Tableau Public.

Open the file E-commerce dashboard.twbx with Tableau. This will load the dashboard so you can interact with the visualizations.

Look at the data

Open ramakrishna.csv (1).xlsx in Excel (or a spreadsheet tool) to see the data used in the dashboard.

Check the notebook

Open Untitled0.ipynb in Jupyter Notebook to see any analysis/code related to data prep or insights (if included).
