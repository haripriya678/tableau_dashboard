Clone the Repository
git clone https://github.com/haripriya678/tableau_dashboard.git

Open the Tableau Dashboard

Install Tableau Desktop or Tableau Public.

Open the file E-commerce dashboard.twbx with Tableau.

This will load the dashboard so you can interact with the visualizations.

Look at the Data

Open ramakrishna.csv (1).xlsx in Excel (or any spreadsheet tool).

This file contains the data used in the dashboard.

Check the Notebook

Open Untitled0.ipynb in Jupyter Notebook.

This may include analysis or code related to data preparation or insights.
